DataRecorder

public class DataRecorder : @unchecked Sendable

센서 데이터를 파일로 기록하는 역할을 담당하는 클래스입니다.

EEG, PPG, 가속도계, 배터리 데이터를 CSV 및 JSON 형식으로 기록하는 기능을 제공하며, 적절한 동시성 안전성을 보장합니다.
