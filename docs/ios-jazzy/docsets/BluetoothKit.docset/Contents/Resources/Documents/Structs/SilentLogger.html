SilentLogger

public struct SilentLogger : BluetoothKitLogger

로그 출력을 완전히 비활성화하는 로거 구현체입니다.

프로덕션 환경이나 로그가 필요하지 않은 상황에서 사용됩니다. 모든 로그 메시지를 무시하여 성능 오버헤드를 최소화합니다.

예시

let bluetoothKit = BluetoothKit(logger: SilentLogger())
