Configuration

  • SensorConfiguration

    센서 데이터 수집 및 디바이스 통신을 위한 구성 설정입니다.

    이 구조체를 사용하여 BluetoothKit의 기본 동작을 사용자 정의할 수 있습니다.

    예시

    // 기본 설정
let defaultConfig = SensorConfiguration.default

// 사용자 정의 샘플링 레이트
let customConfig = SensorConfiguration(
    eegSampleRate: 500.0,
    ppgSampleRate: 100.0,
    deviceNamePrefix: "MyDevice-"
)
